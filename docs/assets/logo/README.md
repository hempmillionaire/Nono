# NONO logo assets

Official NONO branding artwork. Provided by the project; please don't
repaint or recolor these for external use — link to this directory or
copy a file verbatim.

Naming follows the *ink* color, not the background it looks good on.

| File | Description | Use |
| ---- | ----------- | --- |
| `nono-wordmark-light.png` | White wordmark + mark on transparent background | Dark backgrounds / GitHub dark mode |
| `nono-wordmark-dark.png` | Black wordmark + mark on transparent background | Light backgrounds / GitHub light mode |
| `nono-mark-light.png` | White circular mark only, transparent | Dark backgrounds |
| `nono-mark-dark.png` | Black circular mark only, transparent | Light backgrounds |
| `nono-mark-on-dark.png` | White circular mark on a solid black square | App-icon contexts that need a filled square |

The README uses an HTML `<picture>` element so GitHub serves the dark
variant in light mode and the light variant in dark mode automatically.

Logos are released under the same BSD 3-Clause license as the rest of
the repository, so derivatives are allowed; attribution is appreciated.
